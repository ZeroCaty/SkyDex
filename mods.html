Installation instructions:

Инструкция по установке:

  1. Install the latest Fabric Loader

  2. Download mod archive

  3. Put the contents of the archive into the mods folder in the minecraft directory

  4. Установите последнюю версию Fabric Loader

  5. Скачайте архив с модами

  6. Поместите содержимое архива в папку mods в директории Майнкрафта

List of mods:

Список модов:

  1. DisableCustomWorldsAdvice - removes stupid warnings when starting the world

  2. EntityCulling - optimizes the rendering of objects out of view

  3. Fabric-Api - library for Аabric mods

  4. FerriteCore - optimizes memory usage

  5. Lithium - optimizes many aspects of the game

  6. Phosphor - optimizes the lighting engine

  7. Smoooozie - self-written mod, smoothes the movement of armor stands

  8. SmoothBoot - reduces resource consumption when starting the game

  9. Sodium-Extra - advanced settings for Sodium

  10. Sodium - rendering engine optimization

  11. DisableCustomWorldsAdvice - убирает глупые предупреждения при запуске мира

  12. EntityCulling - оптимизирует рендер объектов вне поля зрения

  13. Fabric-Api - библиотека для Fabric модов

  14. FerriteCore - оптимизирует использование памяти

  15. Lithium - оптимизирует множество аспектов игры

  16. Phosphor - оптимизирует движок освещения

  17. Smoooozie - самописный мод, сглаживает движение стоек для брони

  18. SmoothBoot - уменьшает потребление ресурсов при запуске игры

  19. Sodium-Extra - расширенные настройки для Sodium

  20. Sodium - оптимизация движка рендеринга
